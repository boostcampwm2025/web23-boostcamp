- 기존엔 면접 준비 스터디, 친구들을 모아서 해야하는 불편함이 있다.

## 예시
### [FE] 알림 배지 UI 테스트용 임시 구현 (예시)

사용자 알림 기능에 대한 화면 흐름을 검증하기 위해,
실제 알림 데이터와 무관한 테스트용 알림 배지 UI를 임시로 구현하였다.

또한 알림 개수가 일정 수치를 초과하는 경우(99+),
텍스트가 넘치지 않도록 배지 크기와 정렬을 조정하는 로직을 포함하였다.
해당 구현은 디자인 및 인터랙션 테스트가 목적이며,
추후 실제 알림 API 연동 시 제거 또는 대체될 예정이다.
